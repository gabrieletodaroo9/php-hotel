# PHP Hotel

Un'applicazione web dinamica realizzata in **PHP** che permette di visualizzare e filtrare una lista di hotel in base alle preferenze dell'utente.

Il progetto mette in luce la capacità di integrare logica di programmazione backend all'interno di una struttura HTML, gestendo dati complessi e parametri di ricerca in tempo reale.
